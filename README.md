# SOC Detection Lab

Built a home SOC lab using Elastic Stack, Sysmon, and Windows event logs to detect simulated attacks.

## Tools Used
- Elasticsearch
- Kibana
- Sysmon
- Winlogbeat
- Kali Linux

## Attacks Simulated
- Mimikatz credential dumping
- PowerShell download cradle
- RDP brute force

## Skills Demonstrated
- SIEM deployment
- Threat detection
- Log analysis
- Incident investigation
- KQL queries
